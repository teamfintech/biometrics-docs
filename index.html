Quick start


Overview


    We provide our customer a single javascript bundle. which is portable and pretty simple to use. Users have to import this bundle in any webpage of the web application. Whenever the javascript bundle is imported into a web page, it can be triggered with a submit event. Once the button is clicked, desired Window will open. In the enrollment window, it will automatically capture fingers one by one. The picture of two palms indicates which finger of which hand is to be put on the scanner. It is programatically customizable which fingers you like to capture. It will show the graphical representation of the fingerprint captured in the specific window. The verification window takes one fingerprint. It's generally used to verify existing fingerprint.

    Returning Data Set

    After capturing all the required fingers, the submit button have to be clicked. Once this button is clicked, All of the finger data will be on your web application. Then you can do whatever you want with these data. You can store those data to your database. you can verify the existing fingerprint from database with instantly captured fingerprint from verification window. The resulted dataset holds the image height-width, raw image data, image quality score and ISO formatted data.
